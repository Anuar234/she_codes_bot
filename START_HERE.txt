🚀 КАК ЗАПУСТИТЬ (3 ШАГА):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ШАГ 1: Получить данные для .env
   📍 BOT_TOKEN:     Telegram → @BotFather → /newbot
   📍 CHAT_ID:       Добавь @userinfobot в группу
   📍 OPERATOR_IDS:  Твой ID от @userinfobot

ШАГ 2: Настроить .env
   📝 Открой файл .env (он уже создан!)
   📝 Замени YOUR_BOT_TOKEN_HERE на свой токен
   📝 Замени YOUR_CHAT_ID_HERE на ID группы
   📝 Замени YOUR_OPERATOR_ID_HERE на свой ID

ШАГ 3: Запустить
   pip install -r requirements.txt
   python test_config.py
   python main.py
