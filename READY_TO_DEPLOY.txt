┌─────────────────────────────────────────────────────────────┐

└─────────────────────────────────────────────────────────────┘

1. Запушь код на GitHub:
   git add .
   git commit -m "Ready to deploy"
   git push

2. Зайди на railway.app
3. New Project → Deploy from GitHub
4. Выбери репозиторий
5. Добавь переменные:
   BOT_TOKEN=8235754207:AAE45_ThDDng59ZfGim-gVJPcg5Bhxj2aQk
   CHAT_ID=твой_chat_id
   OPERATOR_IDS=твой_id

⏱️ Время: 3 минуты
💰 Стоимость: Бесплатно (500 часов/месяц)


┌─────────────────────────────────────────────────────────────┐
│ ВАРИАНТ 2: Docker (ЛОКАЛЬНО ИЛИ НА СЕРВЕРЕ) ⭐⭐⭐⭐       │
└─────────────────────────────────────────────────────────────┘

1. Убедись что .env заполнен:
   BOT_TOKEN=8235754207:AAE45_ThDDng59ZfGim-gVJPcg5Bhxj2aQk
   CHAT_ID=твой_chat_id
   OPERATOR_IDS=твой_id

2. Запусти:
   docker-compose up -d

3. Проверь логи:
   docker-compose logs -f

⏱️ Время: 2 минуты
💰 Стоимость: Бесплатно


┌─────────────────────────────────────────────────────────────┐
│ ВАРИАНТ 3: VPS (ПОЛНЫЙ КОНТРОЛЬ) ⭐⭐⭐⭐                  │
└─────────────────────────────────────────────────────────────┘

1. Подключись к серверу:
   ssh user@your-server

2. Выполни:
   sudo apt update
   sudo apt install python3.11 python3-pip git -y
   git clone https://github.com/your-repo/she_codes_bot.git
   cd she_codes_bot
   python3.11 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt

3. Отредактируй .env:
   nano .env
   # Вставь BOT_TOKEN, CHAT_ID, OPERATOR_IDS

4. Запусти:
   python main.py

5. (Опционально) Запусти как сервис:
   См. DEPLOY.md раздел "VPS (Ubuntu)"

⏱️ Время: 10 минут
💰 Стоимость: $5-10/месяц (VPS)

3. Start Command: python main.py
4. Environment Variables:

   CHAT_ID=твой_chat_id
   OPERATOR_IDS=твой_id
5. Create Service

⏱️ Время: 5 минут
💰 Стоимость: Бесплатно
