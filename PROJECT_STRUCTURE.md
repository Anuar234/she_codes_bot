```
│   │
│   └── 📂 utils/                # Утилиты

├── 📄 main.py                   # 🚀 Точка входа - запуск бота

│

    ├── view_stats.py            # Просмотр статистики
    └── reset_database.py        # Сброс базы данных
```

## 📊 База данных (SQLite)

### Таблицы:

- **users** - Участники бота
- **tasks** - Задания
- **daily_tasks** - Отправленные задания (история)
- **answers** - Ответы пользователей
- **points** - История начисления баллов
- **chat_activity** - Дневная активность в чате
- **warnings** - Предупреждения

## 🎯 Основные файлы

### Для запуска:
- `main.py` - Запуск бота
- `.env` - Настройки (токен, ID)

### Для разработки:
- `config.py` - Загрузка настроек
- `database.py` - Все операции с БД
- `bot/handlers/user.py` - Логика для пользователей
- `bot/handlers/operator.py` - Логика для админов
- `bot/utils/scheduler.py` - Автоматическая отправка заданий

### Для управления:
- `test_config.py` - Проверка настроек перед запуском
- `add_task.py` - Интерактивное добавление заданий
- `view_stats.py` - CLI для просмотра статистики
- `reset_database.py` - Сброс/восстановление БД

## 📝 Логи и данные

Создаются автоматически:
- `bot.log` - Логи работы бота
- `data/bot.db` - База данных
- `data/bot.db.backup` - Резервная копия (при сбросе)

## 🔐 Безопасность

Не коммитятся (в .gitignore):
- `.env` - Токены и ID
- `data/bot.db*` - База данных
- `bot.log` - Логи

## 🚀 Быстрый старт

1. Настройте `.env` (см. QUICKSTART.md)
2. Установите зависимости: `pip install -r requirements.txt`
3. Проверьте настройки: `python test_config.py`
4. Запустите: `python main.py`
